知识点

学会安装python开发环境

windows

1、上网搜索”python 官网”，找到windows 版本的python安装包，下载并安装；
2、确认安装的是python 2 而不是python 3;
3、配置python 环境变量，在path后面追加“C:\Python27”;
4、打开命令行，输入python，查看输出信息，判断是否安装成功
// 下面步骤是安装代码编辑器的操作
1、上网搜索”atom 官网”，下载windows 版本的atom安装包，下载并安装；
2、打开atom，试着随便写点东西，命名并保存。
