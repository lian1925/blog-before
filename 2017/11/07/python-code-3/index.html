课3 数字运算 | Python

作者： Lian

日期： 2017年11月07日 10:11

每一种编程语言都会包含进行数学运算的方法。很多程序员喜欢用数学吓唬编程世界外面的人，他们经常撒谎说他们是数学天才，其实他们根本不是。让我们见识一下python的数学运算。

本课将会学习很多数学运算符号，让我们看看他们是什么？你要一边写，一边念，直到自己念烦为止。
+ 加号
- 减号
* 星号，代表乘法
/ 斜杠，代表除法
% 百分号，代表除余
< 小于号
> 大于号
<= 小于等于号
>= 大于等于号

代码练习

接下来的代码练习将让你学会注释。
打开atom,新建一个文件，命名为lesson3.py,输入下面的代码：


print "I will now count my chickens:"
print "Hens", 25 + 30 / 6
print "Roosters", 100 - 25 * 3 % 4
print "Now I will count the eggs:"
print 3 + 2 + 1 - 5 + 4 % 2 - 1 / 4 + 6
print "Is it true that 3 + 2 < 5 - 7?" 10
print 3 + 2 < 5 - 7
print "What is 3 + 2?", 3 + 2
print "What is 5 - 7?", 5 - 7
print "Oh, that's why it's False."
print "How about some more."
print "Is it greater?", 5 > -2
print "Is it greater or equal?", 5 >= -2
print "Is it less or equal?", 5 <= -2

你应该看到的结果

$ python lesson3.py
I will now count my chickens:
Hens 30
Roosters 97
Now I will count the eggs:
7
Is it true that 3 + 2 < 5 - 7?
False
What is 3 + 2? 5
What is 5 - 7? -2
Oh, that’s why it’s False.
How about some more.
Is it greater? True
Is it greater or equal? True
Is it less or equal? False
$

常见问题

1、为什么 / 除法算出来的结果比实际小？
其实不是没算对，而是它把小数部分省略了，试试 7 / 4 与 7.0 / 4.0 比较一下，你就能看出不同了。
