课5 转义字符 | Python

作者： Lian

日期： 2017年11月07日 11:11

在编程中，有一些字符是很难打印出来的，比如单引号'和双引号"。想象你要打印：’I’m a student.’这时python会理解错误，以为单引号是字符串的边界。
为了解决这个问题，Python引入了转义字符。

知识点

我们将会学习常用的转义字符，花点时间记住它们。
\'
\"
\n
\r
\t

代码练习

tabby_cat = "\tI'm tabbed in."
persian_cat = "I'm split\non a line."
backslash_cat = "I'm \\ a \\ cat."

fat_cat = """
I'll do a list:
\t* Cat food
\t* Fishies
\t* Catnip\n\t* Grass
 """

print tabby_cat
print persian_cat
print backslash_cat
print fat_cat

你应该看到的结果

注意你打印出来的制表符,这节练习中的文字间隔对于答案的正确性是很重要的。
$ python ex10.py
I’m tabbed in.
I’m split
on a line.
I’m \ a \ cat.
I’ll do a list:

   * Cat food
   * Fishies
   * Catnip
   * Grass

$

加分项

1、将转义字符抄在卡片上，记住它们的作用。
2、记得格式化字符串 %r 和%s吗？配合转义字符打印字符串，比较一下它们的区别。

常见问题

1、为什么使用%r，转义字符失效？
因为%r将打印原始的字符串。如果需要转义效果，使用%s。记住：%r用于debug，%s用于显示。
