布尔表达式练习

判断每一题的布尔值。

  1. True and True
  2. False and True
  3. 1 == 1 and 2 == 1
  4. “test” == “test”
  5. 1 == 1 or 2 != 1
  6. True and 1 == 1
  7. False and 0 != 0
  8. True or 1 == 1
  9. “test” == “testing”
    10.1 != 0 and 2 == 1
    11.”test” != “testing”
    12.”test” == 1
    13.not (True and False)
    14.not (1 == 1 and 0 != 1)
    15.not (10 == 1 or 1000 == 1000)
    16.not (1 != 10 or 3 == 4)
    17.not (“testing” == “testing” and “Zed” == “Cool Guy”)
    18.1 == 1 and not (“testing” == 1 or 1 == 0)
    19.”chunky” == “bacon” and not (3 == 4 or 3 == 3)
    20.3 == 3 and not (“testing” == “testing” or “Python” == “Fun”)

技巧

所有的布尔逻辑表达式都可以用下面的简单流程得到结果:

  1. 找到相等判断的部分 (== or !=),将其改写为其最终值 (True 或 False)。 2. 找到括号里的 and/or,先算出它们的值。
  2. 找到每一个 not,算出他们反过来的值。
  3. 找到剩下的 and/or,解出它们的值。
  4. 等你都做完后,剩下的结果应该就是 True 或者 False 了。

示例：
下面我们以 #20 逻辑表达式演示一下:
3 != 4 and not (“testing” != “test” or “Python” == “Python”)
接下来你将看到这个复杂表达式是如何逐级解为一个单独结果的:

  1. 解出每一个等值判断:
  2. 3 != 4 为 True: True and not (“testing” != “test” or
           "Python" == "Python")
  3. “testing” != “test” 为 True: True and not (True or
           "Python" == "Python")
  4. “Python” == “Python”: True and not (True or True)
  5. 找到括号中的每一个 and/or :
  6. (True or True) 为 True: True and not (True)
  7. 找到每一个 not 并将其逆转:
  8. not (True) 为 False: True and False
  9. 找到剩下的 and/or,解出它们的值:
  10. True and False 为 False
    这样我们就解出了它最终的值为 False.
