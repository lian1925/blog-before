课2 注释和#号 | Python

作者： Lian

日期： 2017年11月07日 09:11

在编程的世界里，注释是很重要的。

他们可以用自然的语言告诉你，某段代码的功能是什么
当你不需要某些代码时，你可以用注解的方式将它们临时禁用。

代码练习

接下来的代码练习将让你学会注释。
打开atom,新建一个文件，命名为lesson2.py,输入下面的代码：

# 这是单行注释，你可以加上自己代码的说明
print "I could have code like this." # 当然你也可以在代码后面加注释哈
# 这是多行注释
# python 会忽略#(井号)后面的内容
print "This will run."

你应该看到的结果

$ python ex2.py
I could have code like this.
This will run.
$

加分项

1、弄清楚”#”符号的作用，而且记住它的名字。
2、朗读自己写的每一段代码。

常见问题

1、为什么 print “Hi # here” #后面的内容没有被python 忽略？
这行代码中的#在字符串内部，所以它被当成字符串中的普通字符，而不代表注解的意思。
