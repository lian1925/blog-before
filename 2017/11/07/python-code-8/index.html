课8 读取文件 | Python

作者： Lian

日期： 2017年11月07日 14:11

本节将学习python 读取文件的操作。
注意了。读写文件是一件非常谨慎的活，一旦你不小心的话，很有可能会将有用的文件弄坏或清空。

知识点

关于读取文件，我们将学习下面两个新命令。
open(filename)
read()

代码练习

本节我们需要准备两个文件，一个是正常的txt文件，命名为lesson8_data.txt，写入如下内容：
Hello python.

另一个文件命名为lesson8.py，填写如下代码：

from sys import argv
script, filename = argv 4
txt = open(filename)
print "Here's your file %r:" % filename
print txt.read()
print "Type the filename again:"
file_again = raw_input("> ")

txt_again = open(file_again)

print txt_again.read()

你应该看到的结果

$ python lesson8.py lesson8_data.txt
Here’s your file ‘lesson8_data.txt’:
This is stuff I typed into a file.
It is really cool stuff.
Lots and lots of fun to have in here.
Type the filename again:
> lesson8_data.txt
This is stuff I typed into a file.
It is really cool stuff.
Lots and lots of fun to have in here.
$

加分项

1、逐行阅读代码，试着从后往前读，理解代码
2、为每行代码添加注释

常见问题

1、txt = open(filename) 返回的是文件的内容吗?
不是,它返回的是一个叫做“file object”的东西,你可以把它想象成一个磁带机或者 DVD 机。你 可以随意访问内容的任意位置,并且去读取这些内容,不过这个 object 本身并不是它的内容。
