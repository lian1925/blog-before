课10 认识函数 | Python

作者： Lian

日期： 2017年11月07日 14:11

在编程中，函数是非常重要的概念。程序员都喜欢函数，因为函数可以用来给代码块命名，方便代码复用。

函数可以做三件事：
1、它们给代码片段命名，就像变量给数字和字符串命名一样；
2、它们可以接收参数，就像你的脚本可以接收参数一样；
3、它们可以返回参数。

知识点

函数定义：
1、定义以 def 开始
2、接着是函数名
3、接着是括号，括号内包含参数
4、接着是冒号：
5、下一行的函数体，使用4个字符的缩进
6、函数结束位置，取消缩进

函数调用
1、使用函数名称
2、接着是括号，括号内包含参数，多个参数使用逗号隔开

代码练习

def add(a, b):
    print "ADDING %d + %d" % (a, b)
    return a + b
def subtract(a, b):
    print "SUBTRACTING %d - %d" % (a, b)
    return a - b
def multiply(a, b):
    print "MULTIPLYING %d * %d" % (a, b)
    return a * b
def divide(a, b):
    print "DIVIDING %d / %d" % (a, b)
    print "Let's do some math with just functions!"
    return a / b
age = add(30, 5)
height = subtract(78, 4)
weight = multiply(90, 2)
iq = divide(100, 2)

print "Age: %d, Height: %d, Weight: %d, IQ: %d" % (age, height, weight, iq)

# A puzzle for the extra credit, type it in anyway.
print "Here is a puzzle."

what = add(age, subtract(height, multiply(weight, divide(iq, 2))))

print "That becomes: ", what, "Can you do it by hand?"

常见问题

1、为什么 Python 会把函数或公式倒着打印出来?
其实不是倒着打印,而是自内而外打印。如果你把函数内容逐句看下去,你会发现这里的规律。试 着搞清楚为什么说它是“自内而外”而不是“自下而上”。
