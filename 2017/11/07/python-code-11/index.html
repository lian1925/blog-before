课11 知识回顾 | Python

作者： Lian

日期： 2017年11月07日 14:11

这节课，将不会有新知识，也没有任何代码。我将会让你完成一个表格，帮助你回顾直到现在你所学到的所有知识。

首先，回到我们第一次的代码练习，将你碰到的每一个新的知识点写下来。确保你的知识点列表是完整的。

下一步，在每个知识点后面写出它的名字，并且说明它的作用。如果你忘记了它的作用，可以结合当时的脚本，理解其作用。或者上网找一下。

你或许会碰到一些横竖找不到答案的东西，只要把这些记在列表里，它可以提示你还有哪些知识不懂，等下次碰到的时候，你就不会轻易跳过了。

你的列表做好之后，再花几天时间重写一遍这份列表，确认里面的东西都是正确的。你可能觉得很无聊，不过你还是需要坚持完成任务。

等你记住了列表的所有内容，就试着把这份列表默写一遍。如果发现自己忘记了哪些内容，就回去再记一遍。

这种知识梳理和记忆是枯燥无味的，但是它的意义非常重要。它会让你真正的掌握所学的知识，明确自己努力的目标。
