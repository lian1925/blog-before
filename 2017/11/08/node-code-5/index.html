课5 安装MySQL | Node.js

作者： Lian

日期： 2017年11月08日 17:11

本节将会学习如何安装MySQL数据库。

Linux

如果你的系统是CentOS,则安装方法如下：
1、获取MySQL资源

wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm
sudo rpm -ivh mysql-community-release-el7-5.noarch.rpm
yum update

2、安装

sudo yum install mysql-server
sudo systemctl start mysqld

3、登录数据库

mysql -u root -p

4、退出数据库

exit

如果你的系统是Ubuntu,则安装方法如下：
1、更新apt-get包管理器
sudo apt-get update
2、安装mysql
sudo apt-get install mysql-server-5.5
3、配置mysql
sudo mysql_secure_installation
4、查看mysql 版本
mysql –version
5、查看mysql当前状态
service mysql status
6、运行mysql
mysql –uroot -p
