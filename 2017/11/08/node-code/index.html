这节课不需要编程，它的主要目的是让你学会在自己的电脑安装Node.js开发环境。

知识点

1、学习上网搜索资料
2、安装开发环境

Windows

1、上网搜索“Node.js 官网”，找到windows版本的安装包，下载并安装；
2、官网提供LTS和Current两种版本的安装包，建议选择LTS;
3、打开命令行，输入node -v，查看输出信息，判断Node.js是否安装成功；
4、接着，输入命令npm -v，判断npm是否安装成功;

Mac OSX

1、上网搜索“Node.js 官网”，找到Mac版本的安装包，下载并安装；
2、官网提供LTS和Current两种版本的安装包，建议选择LTS;
3、打开 terminal，输入node -v，查看输出信息，判断Node.js是否安装成功；
4、接着，输入命令npm -v，判断npm是否安装成功;

Linux

如果你的系统是Debian 或者 Ubuntu，安装方法如下：
1、打开terminal,使用如下命令安装，

curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs

2、等待安装完毕，输入如下命令，检查是否安装成功

node -v
npm -v

如果你的系统是 Centos 或者 Fedora,安装方法如下：
1、打开terminal,使用如下命令安装，

curl --silent --location https://rpm.nodesource.com/setup_6.x | sudo bash -

2、等待安装完毕，输入如下命令，检查是否安装成功

node -v
npm -v

常见问题

1、为什么建议用户到软件官网下载安装包？
虽然网上有各式各样的安装包，不过鱼龙混杂，有经验的程序员喜欢去官网下载，这样可以保证下载到的安装包是最新版本，最稳定的。
2、LTS 和Current版本有何区别？
LTS 是”Long Term Support”,长时间支持版本，代表当前性能最稳定的版本。
Current 是当前发布的最新版本，一些新特性没有经过全面测试，可能存在不稳定的风险。
