前面学习完Node.js环境的安装后，本课将会学习如何编写第一个程序，并让它运行起来。

Node.js最重要的功能是作为后端服务器开发。那么，我们的第一个程序就是构建一个简单web服务器。

代码练习

打开atom，新建一个文件，命名为app.js，敲打下面的代码。这个命名需要注意，因为所有的Node.js代码最好都以”.js”为后缀。

const http = require('http');

const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World\n');
});

server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});

OK，代码编写完，打开命令行终端，输入内容：
node app.js
注意：请先用命令行进入app.js所在的目录，若是不懂操作，尝试上网搜索

你应该看到的结果

打开浏览器，访问链接http://localhost:3000，可以看到页面返回"Hello World”

加分项

1、修改网页的返回内容，如：Hello Lian(你的名字)
2、修改端口值，如：3000-》4000
