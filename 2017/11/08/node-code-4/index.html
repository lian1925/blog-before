作者： Lian

Express 是一个基于 Node.js 平台的极简、灵活的 web 应用开发框架，是当前Node.js后端开发最流行的框架。本节将学习使用express构建最简单的网站。

代码练习

1、创建一个目录lesson4,并进入目录

mkdir lesson4
cd lesson4

2、创建package.json文件
npm init

3、安装express
npm install express –save

打开atom，新建文件，命名为app.js，放入lesson4目录，编写代码如下：

var express = require('express');
var app = express();

app.get('/', function (req, res) {
  res.send('Hello World!');
});

var server = app.listen(3000, function () {
  var host = server.address().address;
  var port = server.address().port;

  console.log('Example app listening at http://%s:%s', host, port);
});

OK，代码编写完，打开命令行终端，输入内容：
node app.js

你应该看到的结果

打开浏览器，访问链接http://localhost:3000，可以看到页面返回”Hello World”
