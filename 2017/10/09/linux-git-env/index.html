Linux安装Git环境

作者： Lian

日期： 2017年10月09日 15:10

目标：为Linux服务器安装Git环境，配置安全协议SSH

安装Git

sudo apt-get update
sudo apt-get install git

查看版本，检测是否安装成功

git --version

生成SSH KEY，默认安装(若有选择，按回车继续)

ssh-keygen -t rsa -b 4096 -C lian1925@126.com

启动 ssh-agent:

eval $(ssh-agent -s)

为 ssh-agent 添加 SSH key

ssh-add ~/.ssh/id_rsa

获取 SSH KEY内容，填写到GitLab账号

$ pbcopy < ~/.ssh/id_rsa.pub
# Copies the contents of the id_rsa.pub file to your clipboard
